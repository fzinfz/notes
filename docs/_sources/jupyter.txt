Server Setup
============

http://jupyter-notebook.readthedocs.io/en/latest/public\_server.html

.. code:: python

    jupyter notebook --generate-config
    
    vi ~/.jupyter/jupyter_notebook_config.py   # c.NotebookApp.password
    
    jupyter console # start the ZMQ-based console
    jupyter console --existing # If no argument specified, guess most recent

conda env
=========

.. code:: python

    anaconda login
    anaconda whoami
    conda create --name=py27 python=2.7
    conda create --name py35 --clone root
    conda remove --name py35 --all
    conda env export > py35.yml
    conda env create -f py35.yml
    
    conda env upload --file=py35.yml
    
    conda info
    
    conda env list

ipykernel
=========

http://ipython.readthedocs.io/en/stable/install/kernel\_install.html

.. code:: python

    jupyter-kernelspec list

.. code:: python

    python3 -m pip install ipykernel
    python3 -m ipykernel install --user
    
    python2 -m pip install ipykernel
    python2 -m ipykernel install --user

virtualenv
==========

.. code:: python

    yum install python34-setuptools
    easy_install-3.4 pip
    
    virtualenv ENV  
    cd ENV  
    source ./bin/activate  
    deactivate
